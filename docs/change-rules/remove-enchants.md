# Remove Enchants

## Remove Enchants <a href="#remove-enchants" id="remove-enchants"></a>

Similar to `delete-enchants` but ignore the enchantment levels.

```yaml
real-changes:
  remove-enchants:
    - 'KNOCKBACK'
```

## Remove all enchantments <a href="#remove-all-enchantments" id="remove-all-enchantments"></a>

```yaml
real-changes:
  remove-all-enchants: true
```

## Remove Stored Enchants <a href="#remove-enchants" id="remove-enchants"></a>

Similar to `delete-stored-enchants` but ignore the enchantment levels.

```yaml
real-changes:
  remove-stored-enchants:
    - 'KNOCKBACK'
```

## Remove all enchantments <a href="#remove-all-enchantments" id="remove-all-enchantments"></a>

```yaml
real-changes:
  remove-all-stored-enchants: true
```
