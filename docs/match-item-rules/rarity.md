# Rarity

## Rarity (1.20.5+) <mark style="color:red;">- Premium</mark>

Require item must have specified rarity. Rarity require you extra set in item, by default all items don't have rarity, for item don't have custom rarity, plugin will use "NONE" as result.

{% hint style="info" %}
This is just mean vanilla item's ratity, if you mean ratity/tier from custom plugins, please view [this page](items.md).
{% endhint %}

```yaml
match-item:
  rarity: NONE
```
